### 2.2 Translate Sentences into Logic Formulas

- $q$ = "I will pass the course"
- $r$ = "The professor accepts bribes"

2. $r \rightarrow \neg p$

3. $\neg r \wedge p \wedge q$

4. $p \rightarrow (r \wedge q)$
